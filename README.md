# 2026CT-BioMech-Jonathan.I-Daniel.L-Bryan.H

# Introduction
This project aims to design and prototype a prosthetic device specially for Aaron, a passionate bushwalker and rock climber who lost his right forearm due to a traumatic accident. The prosthetic aims to restore his ability to climb, in some extent, and perform essential daily tasks.

# Components
- Arduiono
- Arduino cable
- Galvanic skin response
- Vibration motor
- 3-axis accelerometer
- Servo motors
